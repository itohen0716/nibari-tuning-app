# 二上り調弦練習 PWA Ver.1.0

# 絲庵 ─ Shamisen ─

三味線の練習を支えるアプリ。

調弦だけではなく、
耳を育て、
音を楽しみ、
毎日の練習に寄り添うことを目指しています。

---

## 絲庵 開発理念

・小さな一歩を積み重ねる。

・急がない。

・分かることを大切にする。

・手仕事のように、
　一つずつ丁寧に育てる。
